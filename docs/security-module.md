python# Módulo security

## Descripción
El módulo security centraliza la autenticación y la gestión de usuarios del sistema.

## Componentes incluidos
- Modelo de usuario personalizado basado en AbstractBaseUser
- UserManager con validación por correo electrónico
- Vistas de login y cierre de sesión
- Plantillas base, login e inicio
- Registro del modelo en el panel de administración

## Funcionalidades previstas
- Inicio de sesión con correo y contraseña
- Cierre de sesión seguro
- Acceso protegido a páginas internas
- Gestión de usuarios desde Django Admin
